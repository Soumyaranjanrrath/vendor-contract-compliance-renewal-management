# 📖 Project Overview

Vendor Contract Compliance & Renewal Management is a custom ServiceNow scoped application designed to simplify vendor contract administration by automating contract renewal processes and compliance tracking.

The application enables organizations to:

- Manage vendor contracts efficiently
- Monitor contract expiry and renewal dates
- Automate renewal task creation
- Send reminder email notifications
- Process renewal decisions
- Track compliance status
- Generate reports and dashboards
- Secure data using role-based access control

---

# 🎯 Business Problem

Organizations often manage vendor contracts manually using spreadsheets or emails, making it difficult to track renewal dates, compliance requirements, and contract ownership.

This application solves these challenges by automating the vendor contract lifecycle using ServiceNow Business Rules, Flow Designer, Notifications, Reports, and ACLs.
