# Vendor Contract Compliance & Renewal Management

## Project Overview

Vendor Contract Compliance & Renewal Management is a custom ServiceNow application developed to automate vendor contract lifecycle management. The application helps organizations track contracts, monitor renewal dates, enforce compliance, automate renewal workflows, and manage role-based access.

---

## Features

- Vendor Contract Management
- Automated Contract Number Generation
- Business Rule Validation
- Renewal Status Automation
- Flow Designer Automation
- Vendor Renewal Task Creation
- Email Notifications
- Reports & Dashboard
- Role-Based Access Control (ACL)
- Renewal Decision Processing
