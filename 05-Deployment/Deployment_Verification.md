# Phase 5.3 – Deployment

## Objective

The objective of this phase was to package the application for migration to higher ServiceNow environments using an Update Set.

---

## Deployment Method

The Vendor Contract Compliance & Renewal Management application was developed within a scoped application.

A ServiceNow Update Set was used to capture all application configurations, including:

- Custom Tables
- Dictionary Entries
- Business Rules
- Flow Designer Flows
- Email Notifications
- Access Control Lists (ACLs)
- Reports & Dashboard
- Application Menu and Modules

---

## Deployment Steps

1. Verified all development changes were captured in the scoped application Update Set.
2. Reviewed Customer Updates.
3. Completed the Update Set.
4. Exported the Update Set as an XML deployment package.
